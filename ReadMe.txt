PLANETSIDE 2 - BM TOOLS - BY BOULETMARC


This tool can be used to fix LaunchPad.exe error of PlanetSide2
OR it can modify the .pack contents of the game, to play with a modded game.

########################################################################################################

PATHS SETUP :

Please set your Game Path is its not already set.
IF YOU WANT to get mods, please set your mods folder location.

########################################################################################################

HOW TO SETUP MODS FOLDER :
- Each folder inside the Mods folder, should have the .pack name
- Each folder inside the Mods folder, should includes all of the unpacked .pack files (Modified Files)

EXEMPLE OF MODS FOLDER:
--> Mods/Assets_169/Experience.txt 				(Modified File)
--> Mods/Assets_169/XXX_Any_Other_Files 		(Unmodified Files)

--> Mods/Assets_238/HUDIndicatorSettings.txt	(Modified File)
--> Mods/Assets_238/XXX_Any_Other_Files 		(Unmodified Files)

########################################################################################################

HOW TO USE MODS :

1. Clic 'Add Mods' if your mods is not already added.
2. Launch the LaunchPad
	ONCE you see the 'Play Game' button on the LaunchPad
4. Select 'move .pack'
5. Clic 'Play Game' on the LaunchPad
	ONCE the game is loading
6. Clic 'move back .pack'

########################################################################################################

REMOVING MODS :

You can remove mods simply by clicking 'Remove Mods' Button.
